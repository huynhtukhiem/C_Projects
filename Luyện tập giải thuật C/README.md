## Giới thiệu về repo
Đây là repo để lưu các kiến thức về ngôn ngữ lập trình C trong quá trình học tập của mình, trong repo này bao gồm các kiến thức sau:
- Các cấu trúc dữ liệu: Mảng 1 chiều, mảng 2 chiều, danh sách liên kết đơn, danh sách liên kết đôi,
- Các giải thuật liên quan đến kiểu dữ liệu String (Chuỗi ký tự)
- Các kiến thức liên quan đến kiểu dữ liệu Struct
